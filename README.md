# TaskManager_stpr

Веб-сервис для управления учебным расписанием школ, колледжей, университетов и других учебных заведений.  
Позволяет создавать, редактировать и просматривать учебные дни и задания в удобном визуальном формате.

---

## Реализованная функциональность

- Отображение расписания по неделям (числитель / знаменатель)
- Добавление учебных предметов в конкретный день (только старостой)
- Создание заданий к предметам (только старостой)
- Указание даты сдачи задания
- Возможность задать количество пар / уроков до сдачи
- Отображение информации о выбранном предмете
- Управление заметками пользователя
- Навигация по календарным периодам
- Создание учебной группы
- Подключение студентов к существующей группе
- Разграничение ролей (староста / участник)
- Индивидуальное отслеживание прогресса выполнения заданий

---

## Основная идея проекта

Основная идея сервиса — объединить расписание и систему контроля учебных заданий внутри учебной группы.

Сервис позволяет централизованно управлять расписанием и дедлайнами, при этом сохраняя индивидуальный контроль выполнения для каждого студента.

---

## Ролевая модель

### Староста группы

- Создаёт группу
- Формирует и редактирует расписание
- Добавляет и изменяет предметы
- Добавляет и редактирует задания
- Устанавливает сроки сдачи
- Определяет количество пар / уроков до сдачи

### Участники группы

- Подключаются к группе
- Просматривают расписание
- Видят задания и дедлайны
- Отмечают выполнение заданий для себя

Только староста имеет право изменять расписание и добавлять задания.  
Участники не могут изменять структуру расписания или дедлайны.

---

## Логика работы с заданиями

Задания создаются старостой и становятся доступными всей группе.

Каждое задание может иметь:

- Конкретную дату сдачи
- Либо количество оставшихся пар / уроков до сдачи

### Индивидуальный прогресс

Каждый студент может:

- Отмечать задание как выполненное
- Отслеживать собственный прогресс
- Видеть статус выполнения только для себя

Таким образом:

- Расписание и задания — общие для группы  
- Прогресс выполнения — персональный  

Сервис помогает управлять учебной нагрузкой и контролировать сроки сдачи в рамках группы.

# Установка

## 1. Установка зависимостей
